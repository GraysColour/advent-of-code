Small "framework" for implementing solutions to Advent of Code.

Usage:

Create a package called "YearYYYY" where YYYY are the 4 digits of the year, for example "Year2022". Do not sub-package it, but ensure it's created at the root.

Create a class for each day of the puzzles, and name them "DayZ" where Z is the digit of the day, for example: Day7.java.

Ensure the class

  • extends DayX
  • Has a main method that instatiates the class and calls: doIt();
  • A constructor that passes handling of command line arguments
  • An implementation for a solver for part 1
  • Optionally an implementation for a solver for part 2

For example:


        package Year2022;

        import absbase.DayX;
        import util.Result;

        public class Day7 extends DayX {

          public static void main(String[] args){
            new Day7(args).doIt();
          }

          public Day0(String[] args) {
            super(args);
          }

          public Result solve(String input) {
            ...
            return Result.createResult(...the result...);
          }

          Result solvePart2(String input) { // optional
            return Result.createResult(0L);
          }
        }
